# MoravecITprofil

Krátký osobní profil, který dynamicky načítá sekce z `profile.json` pomocí `app.js`.

Jak spustit lokálně:

```bash
python3 -m http.server 8000
# otevřít http://127.0.0.1:8000/
```

Git workflow a Conventional Commits jsou použitá v historii commitů.

Deploy na GitHub Pages:

1. Pushtě na `main`.
2. V repozitáři GitHub -> Settings -> Pages -> Source: `main` branch, `/` root.
